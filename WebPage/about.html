About project

This project is build as a class project for Visual Analitycs class taught at "Universidad de los Andes" by John Alexis Guerra and Jose Tiberio Hernandez as a Master class for Systems and Computation Engineering Department.

The goal of the course and project is, given a real client with a need and a specific data, to build visualitations that allows the user and client to get insights from it.
